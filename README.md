# 🎬 Dogflix - Movie Reviews & Trailers

[English](#english) | [Español](#español)

---

## English

### 🚀 Description
Dogflix is a React-based web application where users can explore movie reviews, check ratings, and watch trailers. The project focuses on interactive UI/UX and complex data filtering on the client side.

### ✨ Key Features
* **Advanced Filtering:** Filter movies by year, genre, and rating.
* **Favorites System:** Users can mark movies as favorites.
* **Video Integration:** Embedded trailers for an immersive experience.
* **Local Data Management:** Efficient handling of mock data through a structured JavaScript architecture.

### 🛠️ Technologies
* **Frontend:** React.js, HTML5, CSS3 (Custom Layout).
* **State Management:** React Hooks (`useState`, `useEffect`).
* **Data:** JavaScript Objects / Mock Data.

## Español

### 🚀 Descripción
Dogflix es una aplicación web creada con React donde los usuarios pueden explorar reseñas de películas, ver calificaciones y reproducir trailers. El proyecto pone énfasis en una interfaz interactiva y en el filtrado complejo de datos en el lado del cliente.

### ✨ Funcionalidades Clave
* **Filtrado Avanzado:** Búsqueda de películas por año, género y calificación.
* **Sistema de Favoritos:** Permite a los usuarios marcar y guardar sus películas preferidas.
* **Integración de Video:** Trailers embebidos para una experiencia completa.
* **Manejo de Datos Local:** Gestión eficiente de información mediante una arquitectura de objetos JavaScript (Mock Data).

### 🛠️ Tecnologías utilizadas
* **Frontend:** React.js, HTML5, CSS3 (Maquetación personalizada).
* **Gestión de Estado:** React Hooks (`useState`, `useEffect`).
* **Datos:** Objetos JavaScript / Mock Data.
